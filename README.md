# Atom Erlang Build package

The Erlang-Build package uses the [Rebar](https://github.com/rebar/rebar) build
tool, so your program must use rebar.

## Features

 * Compile rebar appliations on project save
 * auto run werl

## Settings
 * rebar path
 * werl path
